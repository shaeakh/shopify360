# Shopify360 🛍️

A modern Flutter-based e-commerce application that provides a seamless shopping experience.

![App Demo](assets/images/app-demo.gif)

## 🌟 Features

- 🔐 User Authentication
- 🏠 Beautiful Landing Page
- ⏱️ Timeline View
- 📱 Responsive Design
- 🎨 Material Design 3.0
- 🔄 State Management with Riverpod
